# Cook With Me

A Django-based recipe management web application with full CRUD operations.

## Features
- View all recipes
- Add new recipes
- Edit existing recipes
- Delete recipes
- Recipe categories (Breakfast, Lunch, Dinner, Snack, Dessert)

## Tech Stack
- Python
- Django
- MySQL
- HTML/CSS

## How to Run
1. Clone the repo
2. Run `pip install -r requirements.txt`
3. Set up MySQL database
4. Update `settings.py` with your database credentials
5. Run `python manage.py migrate`
6. Run `python manage.py runserver`
7. Open `http://127.0.0.1:8000/`
